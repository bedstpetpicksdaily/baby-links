Frequently Asked Questions

Does the Sirona S really need ISOFIX, or can I use the seat belt?

ISOFIX installation is available but not required—the Sirona S can be installed using a standard vehicle seat belt, which makes it compatible with older cars. However, I strongly recommend ISOFIX where your vehicle supports it. The installation is faster, more secure, and the rotation mechanism feels more stable. If you own multiple vehicles, ISOFIX is the feature that justifies the premium price.

How does it compare to the Graco TrioGrow or other rotating options?

The Graco TrioGrow rotates but doesn't offer 360-degree movement—you're limited to a few preset positions. The Sirona S's full rotation is smoother and faster, which matters daily. Build quality on the Cybex feels noticeably sturdier, though the Graco is $200-$300 cheaper. If budget is your priority, the Graco works; if convenience matters more, the Sirona wins.

Will it fit in a standard back seat, or do I need extra space?

It fits standard back seats, but the rotating base does extend further than fixed seats. In our mid-size SUV, it fits comfortably. In a compact sedan, you'll want to test it first or check the dimensions (base footprint is approximately 26 inches wide). The rotating mechanism doesn't fold, so the seat takes permanent space—it's not collapsible like some travel options.
